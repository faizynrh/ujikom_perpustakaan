Tata Cara Instalasi

1. Import database `faiz_ukkperpus` ke MySQL.
2. Unduh Python dari https://www.python.org/downloads/.
   Saat instalasi, centang semua opsi agar Python terdaftar di PATH.
3. Buka file `app\venv\pyvenv.cfg` dan sesuaikan bagian path Python dengan lokasi Python di komputer Anda.
   Contoh:
   Sebelum:
   C:\Users\Lenovo\AppData\Local\Programs\Python\Python313
   Setelah:
   C:\Users\PC\AppData\Local\Programs\Python\Python313
4. Aktifkan environment:
   - Masuk ke path APP.
   - Jalankan: venv\Scripts\activate
5. Jalankan aplikasi dengan perintah: py app.py
6. Akses aplikasi melalui link yang muncul di terminal.
